Horizontal Align
  • horizontally Center Align Elements (margin: auto;) :
    To horizontally center a block element (like <div>) in the container, use margin: auto; .

  • horizontally Center Align Text (text-align: center;) :
    To just center the text inside an element, use text-align: center;

  • horizontally Center an Image :
    To center an image, set left and right margin to auto and make it into a block element.

CSS Horizontal & Vertical Align

注意事項：

  • Center Align Elements (margin: auto;) :
    To horizontally center a block element (like <div>) in the container, use margin: auto; .   Setting the width of the element will prevent it from stretching out to the edges of its container.   The element will then take up the specified width, and the remaining space will be split equally between the two margins.
    Note: Center aligning has no effect if the width property is not set (or set to 100%).


  • Center Align Text (text-align: center;) :
    To just horizontally center the text inside an element, use text-align: center;


  • Center an Image :
    To horizontally center an image, set left and right margin to auto and make it into a block element.

Center Vertically

注意事項：

There are many ways to center an element vertically in CSS.

  • Center Vertically - Using padding :
    • A simple solution is to use top and bottom padding. To center both vertically and horizontally, use padding and text-align: center.

    The text-align property specifies the horizontal alignment of text in an element.
          text-align: center ; To centers the text content <h2> horizontal in an element <div id="header">.

          padding : 25px 0px ; Set the padding- top and bottom property to 25px and padding- left and right property to 0px in the element <div id="header">.

  • Center Vertically - Using line-height :
    • Another trick is to use the line-height property with a value that is equal to the height property.
          line-height: 83px;
          height: 83px;

    The text-align property specifies the horizontal alignment of text in an element.
          text-align: center ; To centers the text content <h2> horizontal in an element <div id="header_lineht">.

    The vertical-align property sets the vertical alignment of an element.
          vertical-align : middle ; The element <h2> is placed in the middle of the parent element<div id="header_lineht">.


  • Center Vertically - Using position & transform :
    • If padding and line-height are not options, another solution is to use positioning and the transform property.
    translateY(y) : Defines a translation, using only the value for the Y-axis.
    translate(x,y) : Defines a 2D translation.

    (一) <div id="header_transY">
          text-align: center; To centers the text content <h2> horizontal in the element <div id="header_transY">.

          position: relative; top: 50%; in the <h2> element.
          transform: translateY(-50%); in the <h2> element.


    (二) <div id="header_transXY">
          position: relative; in the element <div id="header_transXY">.

          position: absolute; top: 50%; left: 50%; in the <h2> element.
          transform: translate(-50%, -50%); in the <h2> element.



  • Center Vertically - Using Flexbox :
    • You can also use flexbox to center things. Just note that flexbox is not supported in IE10 and earlier versions.

          display: flex;
          justify-content: center;
          align-items: center;
